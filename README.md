<h1>osTicket - Konfiguration-nach-der-Installation</h1>
Dieses Tutorial beschreibt die Konfiguration nach der Installation des Open-Source-Helpdesk-Ticketsystems osTicket.<br />


<h2>Environments und verwendete Technologien</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Internet Information Services (IIS)

<h2>Verwendete Betriebssysteme</h2>

- Windows 10</b> (21H2)

<h2>Übersicht der durchgeführten Aktionen im neu installierten System</h2>

- Neuen Support-Mitarbeiter im System hinzugefügt.
- Neue Benutzer/Kunden hinzugefügt.
- Bearbeitung des Tickets, das vom neu hinzugefügten Kunden gesendet wurde.
